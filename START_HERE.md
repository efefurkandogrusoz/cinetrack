## 🚀 BAŞLAMAK İÇİN (5 Dakika)

### 1️⃣ TMDB API Key Al
```
https://www.themoviedb.org/settings/api
→ Ücretsiz hesap oluştur
→ API key kopyala
```

### 2️⃣ .env Dosyasını Oluştur
```bash
copy .env.example .env
```

### 3️⃣ API Key'i Yapıştır
```env
VITE_TMDB_API_KEY=your_api_key_here
```

### 4️⃣ Çalıştır
```bash
npm run dev
```

### 5️⃣ Tarayıcıyı Aç
```
http://localhost:5173
```
